🔒 Politique de confidentialité – Clip Snap

Dernière mise à jour : 11 juillet 2025

Chez Clip Snap, le respect de votre vie privée est une priorité absolue. Cette extension Chrome est conçue pour fonctionner entièrement localement sur votre navigateur, sans transmission de données vers des serveurs tiers.

📁 Données collectées

Clip Snap ne collecte aucune donnée personnelle. Aucune des informations suivantes n’est collectée :

  • Nom, adresse, e-mail, identifiants
  • Informations médicales
  • Données financières
  • Identifiants / mots de passe
  • Messages ou communications privées
  • Adresse IP, GPS, localisation
  • Historique de navigation
  • Contenu des sites visités

⚙️ Fonctionnement de l’extension

Clip Snap permet de capturer manuellement une portion visible de l’écran sur Twitch via sélection souris.

Les captures sont traitées localement via chrome.tabs.captureVisibleTab et enregistrées avec chrome.downloads.

⚠️ Clip Snap ne fonctionne pas sur YouTube. Capture uniquement des images, pas de vidéo.

L’autorisation activeTab est utilisée temporairement pour injecter le script de capture.

🌐 Connexion Internet

Aucun appel à un serveur distant. Le seul lien externe possible mène à la documentation utilisateur.

📦 Stockage des données

Aucun stockage local ni cache : aucune donnée n’est conservée entre les sessions.

👁‍🗨 Suivi & Analyse

Aucun tracker, analyse ou publicité n’est intégré.

⚖️ Conformité RGPD & confidentialité

  • Aucune donnée personnelle
  • Aucun consentement requis
  • Pas de profilage ni traitement automatisé
