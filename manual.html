🎬 Manuel d’utilisation – Clip Snap

🎯 À quoi sert Clip Snap ?

Clip Snap vous permet de capturer une portion précise de l’écran visible (par exemple sur Twitch), en créant automatiquement une image JPG avec un aperçu et un bouton de téléchargement.

🛠️ Installation manuelle (Chrome)

  1. Téléchargez le dossier clip_snap contenant les fichiers (manifest.json, popup.html, etc.).
  2. Ouvrez Google Chrome et allez à : chrome://extensions
  3. Activez le mode développeur (en haut à droite)
  4. Cliquez sur « Charger l’extension non empaquetée »
  5. Sélectionnez le dossier clip_snap
  6. L’extension apparaîtra dans votre barre d’extension Chrome ✅

⚠️ Ne pas modifier les fichiers

Le fonctionnement est optimisé tel quel. Toute modification manuelle peut rendre l’extension instable ou inutilisable.

✨ Utilisation

  • Rendez-vous sur Twitch sur une page contenant un clip
  • Cliquez sur l’icône Clip Snap dans la barre Chrome
  • Cliquez sur le bouton « Sélection manuelle (JPG) »
  • Un curseur en croix apparaît 🖱️
  • Cliquez-glissez pour sélectionner une zone à capturer
  • Un aperçu apparaît avec l’image et deux boutons :
    • 📥 Télécharger pour enregistrer
    • ❌ Annuler pour recommencer
  • L’image est enregistrée dans vos téléchargements sous le nom : clip_snap_timestamp.jpg
